## Featured Work

### Consumer Behavior Analysis Platform
An internal analytics system built around a dummy web application to capture user interaction data and generate **descriptive insights, predictive signals, and prescriptive recommendations** using both real-time and batch pipelines.

The platform supports:
- Understanding *what happened* through behavioral summaries and funnels  
- Anticipating *what is likely to happen* via predictive patterns  
- Informing *what actions to take* using rule-based and model-assisted recommendations  

The project mirrors real product analytics platforms by combining data processing, analytical logic, and system-level design.

**Focus:** Behavioral analytics, decision-oriented insights, analytics pipelines  
**Stack:** Python, data processing workflows, analytics dashboards  

---

### Land Price Estimation using Spatial Feature Analysis
A geospatial prediction system that estimates land prices using **infrastructure proximity, population density, and spatially engineered features** derived from GIS data.

The baseline implementation relies on deterministic, formula-driven spatial logic that performs reliably for estimation tasks.  
Machine learning components are partially implemented, with certain limitations arising from the absence of consistent long-term historical land price data for India. As a result, parts of the ML pipeline remain under refinement and are yet to be fully integrated.

The project currently emphasizes **robust spatial feature design, model-readiness, and decision-support applicability**.

**Focus:** Spatial reasoning, feature engineering, predictive system design  
**Stack:** GeoPandas, Rasterio, Scikit-learn, GIS workflows  

---

Ongoing experimentation on semantic segmentation of satellite imagery for land-use classification. Work includes **model experimentation, preprocessing pipelines, and intermediate deployment considerations**, with attention to performance under constrained computational resources.

**Focus:** Remote sensing workflows, segmentation pipelines, experimentation
