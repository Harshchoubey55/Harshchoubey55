## ⟡ Featured Work

### 🔹 [Realtime Consumer Behavior and Pattern Analysis](https://github.com/Harshchoubey55/Realtime-Consumer-Behavior-and-Pattern-Analysis)
> Behavioral analytics · Real-time + batch pipelines · Decision-oriented insights

An internal analytics system built around a dummy web application to capture user interaction data and generate **descriptive insights, predictive signals, and prescriptive recommendations** using both real-time and batch pipelines.

The platform supports:
- Understanding *what happened* through behavioral summaries and funnels
- Anticipating *what is likely to happen* via predictive patterns
- Informing *what actions to take* using rule-based and model-assisted recommendations

`Python` `TypeScript` `Analytics Pipelines` `Behavioral Data` `ETL` `Dashboards`

---

### 🔹 [Land Price Estimation using Spatial Feature Analysis](https://github.com/Harshchoubey55/Land-Price-Estimation-using-Spatial-Feature-Analysis)
> Geospatial reasoning · Feature engineering · Predictive system design

A geospatial prediction system that estimates land prices using **infrastructure proximity, population density, and spatially engineered features** derived from GIS data. The baseline relies on deterministic, formula-driven spatial logic; ML components are partially implemented, with ongoing refinement due to limited historical land price data for India. Emphasizes **robust spatial feature design, model-readiness, and decision-support applicability**.

`GeoPandas` `Rasterio` `Scikit-learn` `GIS Workflows` `Proximity Modeling`

---

### 🔹 Satellite Image Segmentation *(Ongoing)*
> Remote sensing · Segmentation pipelines · Constrained compute

Ongoing experimentation on **semantic segmentation of satellite imagery** for land-use classification. Work includes model experimentation, preprocessing pipelines, and intermediate deployment considerations, with attention to performance under constrained computational resources.

`PyTorch` `U-Net` `Satellite Imagery` `Semantic Segmentation`

---

### 🔹 [Multilingual News Summarization & Headline Generation](https://github.com/Harshchoubey55/Multilingual-News-Summarization-and-Headline-Generation-Using-Transformers)
> NLP · Transformers · Multilingual text generation

A transformer-based NLP system for automatic **summarization of news articles and generation of concise headlines** across multiple languages. Explores cross-lingual transfer, fine-tuning strategies, and evaluation of generative text quality using modern language models.

`HuggingFace Transformers` `PyTorch` `NLP` `Seq2Seq` `Multilingual Models`
